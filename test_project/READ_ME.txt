    Для установки зависимостей, необходимо выполнить команду в терминале "poetry install",
находясь в директории \CFT_project\test_project.
    Переменные окружения, необходимые для работы приложения, расположены в файле \CFT_project\test_project\.env
    Функции-контроллеры расположены в файле \CFT_project\test_project\app\handlers.py
    Для запуска приложения локально, нужно зайти в директорию \CFT_project\test_project
и выполнить команду в терминале uvicorn app.main:application. После этого главная страница приложения
будет доступна по адресу "http://127.0.0.1:8000/".

В базе данных уже есть 4 пользователя:
    username: "user1"; password: "parol1",
    username: "user2"; password:"parol2",
    username: "user3"; password:"parol3",
    username: "user4"; password:"parol4"

Пользователь "user1" имеет права администратора.
Администратор создается функцией create_admin, которая расположена в CFT_project\test_project\app\__init__.py.
Пользователь с правами администратора имеет возможность совершать CRUD операции.

    Чтобы просмотреть информацию о зарплате, нужно перейти по адресу "http://127.0.0.1:8000/",
перейти по ссылке "Войти", в открывшейся форме заполнить поля "Имя пользователя" и "Пароль" и нажать на кнопку
"Войти". На открывшейся странице нажать на кнопку "Узнать информацию о зарплате"

    Время работы токена устанавливается в файле CFT_project\test_project\.env при помощи переменной ACCESS_TOKEN_EXPIRE_MINUTES.

В файле CFT_project\test_project\app\config переменная TESTING определяет запущено ли приложения в тестовом режиме или нет.
Если переменная TESTING имеет значение True, приложение работает в тестовом режиме, если значение переменной False,
приложение работает не в тестовом режиме.

    Чтобы запустить приложение в тестовом режиме, нужно для начала присвоить переменной TESTING значение True.
После этого нужно, находясь в директории CFT_project\test_project запустить команду в терминале pytest tests\.
Переменная TESTING располагается в CFT_project\test_project\app\config.
Тесты расположены в директории CFT_project\test_project\tests.

    Чтобы запустить приложение через docker, нужно для начала создать образ докера. Для этого необходимо,
находясь в директории CFT_project\test_project, выполнить команду "docker build . --tag название образа".
После этого нужно запустить контейнер, выполнив команду "docker run -p 8000:8000 название образа".
После запуска контейнера, главная страница приложения будет доступна по адресу "http://127.0.0.1:8000/".
